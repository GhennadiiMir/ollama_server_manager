🚀 Ollama Update Available

Choose the command matching your installation method, copy it, and run it on the host machine.

Docker — NVIDIA GPU

docker pull ollama/ollama:latest && docker rm -f ollama && docker run -d --runtime=nvidia -v ollama:/root/.ollama -p 11434:11434 --name ollama ollama/ollama:latest

Docker — AMD GPU (ROCm)

docker pull ollama/ollama:rocm && docker rm -f ollama && docker run -d --device /dev/kfd --device /dev/dri -v ollama:/root/.ollama -p 11434:11434 --name ollama ollama/ollama:rocm

Docker — CPU only

docker pull ollama/ollama:latest && docker rm -f ollama && docker run -d -v ollama:/root/.ollama -p 11434:11434 --name ollama ollama/ollama:latest

Linux — native install

curl -fsSL https://ollama.com/install.sh | sh

macOS — Homebrew

brew upgrade ollama

⚠ The Docker commands use the container name ollama and volume name ollama. Adjust these, the port, and GPU flags to match your setup. The container will be offline briefly during recreation. Release notes ↗
